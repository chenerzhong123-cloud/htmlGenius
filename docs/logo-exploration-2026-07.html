市场观察 / 2026
Linear · 极简动势用一个连续、极少色彩的动态几何体建立高级技术感。可借鉴“单一强记忆形”，不要复制其条带语言。
Vercel · 极端可缩放三角形证明了小尺寸先于叙事。对 htmlGenius，核心轮廓应优先于复杂的 HTML / AI 隐喻。
Raycast · 可拥有的符号强烈的 app icon 容器 + 高对比标记，适用于桌面工具。代价是容易落入“另一款效率工具”的同质化。
Warp / developer tools · 系统感网格、光标、路径和 token 化颜色能传达开发者气质；必须节制，否则会像 terminal theme 而不像品牌。
