## 1. 背景与目标

v0.4 功能已实现并本地验证。本期做一个**阶段性正式版**：把当前"冷灰 AI 工具风"的侧边栏 + 浮工具栏，按 Anthropic 设计规范换成**暖奶油 + 珊瑚红**的编辑风，**隐藏协同入口**（协同等飞书 OAuth 后再开），打包字体，准备上 Chrome Web Store。

**目标：**
- **G1 换肤**：侧边栏 + 浮工具栏按 DESIGN.md 的 token（色/字/圆角/间距/质感）重塑。
- **G2 字体打包**：Inter（正文）+ Cormorant Garamond（标题）打包进扩展，全平台一致、离线可用。
- **G3 隐藏协同**：移除协同配置 UI（mode 下拉/backend/token/username/保存），默认 mode=local；协同代码保留 dormant。
- **G4 商店就绪**：manifest 权限最小化（不加新权限；移除协同期遗留的 backend host_permissions），包体积增量 < 1MB。

**非目标（YAGNI）：**
- 不改 viewer.html（本期只侧边栏 + 工具栏）。
- 不重构侧边栏结构（仍是 指示→列表→actions，只换肤 + 加小品牌头）。
- 不动协同功能代码（只藏 UI 入口）。
- 不做 hover 花活（规范："primary 按 press 加深，其余不变"）。

## 3. 字体打包（方案 B）

- `extension/fonts/Inter-Variable.woff2`（可变字重，覆盖 400/500，~300KB）
- `extension/fonts/CormorantGaramond-400.woff2`、`-500.woff2`（~150KB×2）
- 合计 ~600KB–1MB。
- `extension/sidepanel.css` 顶部 `@font-face` 本地引用（`src: url('fonts/...')`，同源，无需 CSP 放宽）。
- **浮工具栏也要用**：manifest `web_accessible_resources` 声明 `fonts/*.woff2`，content-script 注入的 `@font-face` 用 `chrome-extension://<id>/fonts/...` 引用（不受页面 CSP 拦）。
- 字体来源：Google Fonts（Inter、Cormorant Garamond 均为 OFL 开源协议，可合法打包再分发；LICENSE 文件一并放入 `extension/fonts/`）。

## 4. 组件改动

### sidepanel.html
- **移除** `#config` 整块（`#cfg-mode` 下拉 + `#cfg-backend`/`#cfg-token`/`#cfg-username` + `#cfg-save`）。
- 顶部加 `#header`：`<span class="brand">htmlGenius</span>` + `<span class="mode-badge">本地模式</span>`。
- 保留 `#mode-indicator`、`#presence`、`#annotations`、`#actions`（export + edit 按钮）。

### sidepanel.css
- 全量重写：上表 token 落地；`@font-face` Inter + Cormorant；卡片/按钮/徽章/链接样式；底部 actions 栏珊瑚主 + 奶油次。
- 保留 `.card-acts` hover 行为、`.card a` 链接样式（颜色改珊瑚）。

### content-script.js
- 注入的 `<style>` 里：工具栏底改 `#181715`；`.hg-toolbar button[data-act="comment"]` 珊瑚主按钮；其余按钮深底次；`.hg-hl` 暖黄微调。
- 注入一段 `@font-face`（Inter + Cormorant，指向 `chrome-extension://<id>/fonts/...`）让工具栏文字也用打包字体。
- **行为零改动**（编辑态/批注/overlay/undo 等逻辑不动）。

### manifest.json
- `web_accessible_resources` 增加 `fonts/*.woff2`（对 `<all_urls>` 暴露，供 content-script 引用）。
- **移除** 协同期遗留的 `host_permissions: ["http://localhost/*", "https://YOUR_BACKEND_HOST/*"]`（本地模式不需要、协同入口已隐藏；OAuth 时再加）。→ 商店权限最小化。
- 不加任何新 permission。

### sidepanel.js
- 移除 `loadCfg`/`saveCfg`/`toggleCfgFields` 及配置相关事件监听（UI 已删）。
- 保留 `renderMode`/`renderCards`/`renderPresence`/export/edit 处理；`renderMode` 不再依赖协同配置。
- 默认 mode=local 由 content-script 侧保证（已是默认）。

## 5. 测试策略

- **自动化**：`uv run pytest -v` 全绿（CSS/HTML 改动不应影响 pytest；conftest 直接写 storage，不依赖配置 UI）。若 `test_ui.py` 有选择器命中受影响，最小调整并说明。
- **手动视觉矩阵**（实现后人工过）：
  - 侧边栏：奶油底、珊瑚主按钮、暖卡片、衬线品牌头、本地模式徽章。
  - 卡片：hover 出回复/删除；引文珊瑚左条；评论里 URL 珊瑚色可点。
  - 浮工具栏：深底、珊瑚「批注」、高亮暖黄。
  - 编辑/查看 toggle：珊瑚/次按钮切换文案。
- **字体**：Mac + Windows 各看一次（打包字体应一致）；离线断网仍正常。

## 6. 已知边界

- **OFL 合规**：Inter / Cormorant Garamond 是 OFL 协议，必须随包带 LICENSE 文件（放 `extension/fonts/LICENSE`）。
- **深工具栏在深色页面**：深底工具栏在深色网页上对比可能偏低（可接受；规范允许深产品面）。
- **打包字体回退**：若 woff2 加载失败，`@font-face` fallback 到系统衬线/无衬线（Georgia / -apple-system）。
- **协同 dormant**：UI 隐藏但代码在；mode 默认 local，用户无法触发协同；OAuth 后恢复入口。

## 7. 退出标准（v0.4.1 本地正式版）

- 侧边栏 + 浮工具栏视觉符合 DESIGN.md 暖色编辑风（人工确认）。
- 协同入口完全隐藏；本地模式功能（批注/编辑/undo/删除/复制/链接化）全可用。
- 字体打包、跨平台一致、离线可用、LICENSE 齐全。
- manifest 权限最小（无 backend host_permissions、无新 permission）。
- pytest 全绿；包体积增量 < 1MB。
