三个方案 · 批注入口怎么藏

三个方案都遵循同一套主次:整页默认是编辑(绝对主角),批注藏在次要入口要点才出,头像点开是登录与团队。区别只在中间那张图 ——「批注」这个小入口长什么样、放哪里。

主视图编辑 · 整页都是编辑相关:模式、工具、保存版本、撤销回退
次入口批注 + 一键复制 · 不点不出现,主视图不被打扰
深入口头像 ▾ · 飞书 / Google 登录、邀请码、在线队友(原 7 个账号控件全收这里)
