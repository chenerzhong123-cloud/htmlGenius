**Goal:** 在 v0.1 viewer 上加「contenteditable 实时编辑 + 自动版本 + 内存撤销」,与 v0.1 批注并存,不破坏 AI HTML 结构。

**Architecture:** editor.js(contenteditable + 清理 + 撤销)+ version.js(序列化 + 滚动窗口版本)+ selection-toolbar.js(单一浮工具栏)+ sanitize.js(白名单清洗);后端 versions 表加 html_content + WAL + 事务 + 删版本处理批注。

**Tech Stack:** 沿用 v0.1(Python FastAPI + SQLite + Vanilla JS + Playwright 测试)。
