Token

AI 理解世界的“最小单位”

它是什么？

Token 是 AI 大模型处理文本时的最小语义单元——它不是字符，也不是单词，而是介于两者之间的“语义碎片”。

把 AI 读文章想象成婴儿看世界：婴儿不会一开始就认识“苹果”这个词，而是先看到红色、圆形、光滑等碎片。Token 就是 AI 眼中的这些“碎片”——“苹果”可能被拆成“苹”和“果”两个 token。

因为 AI 的计费、速度、记忆容量全都按 token 算。你给 ChatGPT 发消息，每 1000 个 token 都要钱；模型的“上下文窗口”能装多少 token，决定了它能记住多少对话。

为什么现在这么火？

01

计费模式的改变

2023 年后几乎所有大模型都按 token 计价。GPT-4 的 API 定价是“每百万 token 若干美元”，开发者每天盯着 token 消耗报告，这个词从技术圈进入了商业圈。

02

token 经济学的诞生

“token 优化”成了新的效率竞赛：同样的答案，谁用更少 token 表达清楚，谁就更省钱。这也催生了 Prompt 压缩、上下文缓存等技术。

03

争议：中文真的比英文贵？

中英文 token 化方式不同。一个汉字约等于 1-2 个 token，而一个英文单词可能只占 0.75 个 token。有用户抱怨“中文对话比英文贵 2-3 倍”，这背后就是 token 切割逻辑的差异。

核心原理

01

分词器上场

大模型内置一个分词器（如 BPE、SentencePiece），它会把你的输入“我今天很开心”拆成 [我, 今天, 很, 开心]。

02

每个 token 变成一个向量

拆分后的每个 token 被查一张巨大的“词汇表”，转换成 768 维或更高维的数字向量——这就是 AI 能“看懂”的文字形式。

03

模型处理 token 序列

模型按顺序读取 token 序列，通过注意力机制推断每个 token 之间的关系，最终输出下一个最可能的 token。这也是为什么 AI 是“一个字一个字蹦出来的”。

和 “字符/单词” 有什么区别？

Token字符单词
核心思路语义最小单元书写最小单元语言最小单元
典型场景AI 模型输入输出文本编辑显示查词典
通俗类比乐高积木块原子分子

一句话总结差异 字符和单词是为人定义的，token 是为 AI 的效率定义的——它是一道“翻译层”，把人类语言转换成 AI 能高效消化的格式。

怎么用上它？

OpenAI Tokenizer 工具

OpenAI 官网提供免费的 Tokenizer 工具，你粘贴一段文字就能看到它被切成多少个 token，直观感受“说话的成本”。

写 Prompt 时注意精简

同样的任务描述，用词越精简 token 越少。省 token = 省钱 + 更快响应。把“请你帮我……”改成“帮我……”可能就省了 5 个 token。

Token 是你和 AI 对话的“硬币”——你花的每一句话，都在花它。
