# htmlGenius

> 在任意网页上划词批注，一键把所有意见打包成给 AI 的指令。让 AI 生成的 HTML 越改越准。

🌐 **官网 / Live site**：<https://www.deuce.monster/htmlgenius/>（中 / English 双语，源码见 [`landing/`](landing/)）

你用 AI 生成了一份 HTML，渲染出来总觉得「这里字号大了」「这段措辞要改」「这个按钮该靠右」。光在对话框里描述，AI 总改不到位。htmlGenius 让你**直接在渲染好的网页上**圈出每一处、写下意见，然后**一键**把全部意见整理成一段带精确定位的指令，复制粘贴给 AI 即可逐条修改。

它是一个 Chrome 扩展：装一次，任何网页都能用。数据留在你自己手里。

## 它能做什么

- **划词批注**：选中网页上的任意文字 → 选区上方弹出「批注」→ 写下你的意见。
- **不破坏原网页**：高亮只是一层覆盖，关掉插件页面照旧，不会动到原文结构。
- **批注跟着内容走**：AI 改完产出新版本后，你的批注会自动定位到新位置；如果原文被删，批注会归档保留，不会丢。
- **层层回复**：可以在一条批注下继续讨论，形成讨论线索。
- **一键回灌**：把当前页面所有批注打包成一段结构化指令并复制到剪贴板，粘进 AI 对话框即可。
- **直接编辑（本地文档）**：本地的 HTML 还能直接在页面上改文字、调样式。

## 安装

1. 打开 `chrome://extensions`，右上角开启「**开发者模式**」。
2. 点「**加载已解压的扩展程序**」，选择本仓库的 `extension/` 目录。
3. 打开任意网页，点工具栏的 htmlGenius 图标，侧边栏即打开。

## 怎么用

1. 在网页上选中一段文字 → 选区上方点「**批注**」。
2. 在侧边栏的输入框写下意见 → 按 **Enter** 保存（Shift+Enter 换行，Esc 取消）。
3. 重复，把所有要改的地方都标出来。
4. 点底部的「**一键复制所有评论**」→ 粘进 AI 对话框。

> 在一条批注上悬停，可以**回复**、**编辑**或**删除**（编辑/删除仅限作者本人；删除会连同其下所有回复一起删）。

## 最近更新

### v0.5.1（2026-07-14 · 当前版本）

- **深色 / 浅色主题切换**：侧边栏右上角一键切换。深色为 Nebula 风格，浅色参照 Airtable 设计规范（白底 + 近黑主按钮），随你浏览的网页明暗搭配；浮动工具栏与确认弹窗也会跟随切换。无偏好时跟随系统。
- **中 / 英 / 日 多语言**：跟随浏览器语言，可在头像旁切换并记住选择。
- **评论可编辑**：作者可编辑自己已保存的评论（团队模式经后端 `PATCH /api/annotations/:id`，带作者校验；编辑实时同步给队友）。
- **收起侧边栏即停止**：关掉侧边栏后浮窗、高亮、编辑态立即消失，页面恢复正常浏览，不再被打扰。
- **失效评论一键清理**：当网页内容变动、旧评论找不到原文位置时，这些评论会归档到侧边栏底部，可一键全部删除。
- **修复本地 HTML 版本陈旧**：若 HTML 文件被外部（如另一个 agent）改过，打开插件时看到的是**新版本**而不是带旧评论的旧版本。
- 编辑工具栏升级：加粗 / 斜体 / 下划线 / 删除线可切换高亮、下划线与删除线共存；字号、对齐生效；文字颜色、荧光笔换成标准图标。

### v0.5.0（2026-07-07）

- **多人协同正式开放**：用飞书账号登录后，同一页面的批注与回复可实时共享给队友。
- **身份硬化**：批注作者绑定飞书账号，只能删除自己发的批注，他人无法伪造。
- 侧边栏新增「飞书登录」入口；本地单人用法完全不变。

### v0.4.2（2026-07-06）

- **修复**：本地 HTML 的编辑现在能保存了——关闭或刷新页面后再打开，会自动恢复你上次的编辑。
- **修复**：Ctrl+Z 撤销不再让批注 / 格式工具栏失灵。

### v0.4.1（2026-07-06）

- 界面换新：侧边栏与浮工具栏采用更清爽的暖色风格。
- 批注的创建、回复、删除全部改在侧边栏里直接完成，**不再弹出浏览器对话框**；Enter 保存、Esc 取消。
- 自带字体，无需联网加载。

## 已知限制

- **回灌需手动粘贴**：复制指令后要自己粘进 AI 对话框（暂不会自动改网页）。
- **远程网页的编辑是临时的**：刷新或关闭页面即丢失，无法存回原网站。
- **协同需登录**：多人实时批注需用飞书账号登录自托管后端；本地单人用法不受影响。
